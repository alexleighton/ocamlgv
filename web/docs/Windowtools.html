Module Windowtools


module Windowtools: sig .. end
Utility module for drawing interactive graphs.
val linelist : Vector.t list -> unit
Draws a line through a list of vectors.
vlist : The list of vectors.
val arrow_line : Vector.t list -> unit
val curvelist : Vector.t list -> unit
Draws a curved line by taking the point list 4 points at a time. If p,p1,p2,p3 are the points considered, it draws a cubic Bezier spline from p to p3 with p1,p2 as control points. The list of points should look like [p;cp1;cp2;p1;cp3;cp4;p2 ... pn] where cp are control points.
vlist : The list of points.
val get_textbox : string -> int * int
Returns the width and height of a box enclosing the given text.
str : The string to get the width and height of.
val get_textloc : string -> Vector.t -> int * int * int * int
Get the x,y location to draw the given string at, in order for it to be centered on the given point.
Returns (x,y,w,h) where x,y are the location to draw the string at and w,h are the width and height of the string if it were drawn.
str : The string to work on.
v : The center point of the string.
val fillcircle : int -> int -> int -> unit
Fills a white circle, then draws a black-outlined circle on top.
x : The x coordinate of the center of the circle.
y : The y coordinate of the center of the circle.
r : The radius of the circle.
val fillellipse : int -> int -> int -> int -> unit
Fills a white ellipse, then draws a black-outlined ellipse on top.
x : The x coordinate of the center of the ellipse.
y : The y coordinate of the center of the ellipse.
rx : The radius in the x direction.
ry : The radius in the y direction.
val fillrect : int -> int -> int -> int -> unit
Fills a white rectangle, then draws a black-outlined rectangle on top.
x : The x coordinate of the lower-left corner of the rectangle.
y : The y coordinate of the lower-left corner of the rectangle.
w : The width of the rectangle.
h : The height of the rectangle.
