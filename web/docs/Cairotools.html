Module Cairotools


module Cairotools: sig .. end
Utility module for drawing using the Cairo library
val pi : float
The constant PI.
val wrap_context : Cairo.t -> (unit -> 'a) -> unit
Saves the current context and then restores it after applying the given function. Requires a function that takes unit as a parameter.
context : The Cairo context to save and restore.
f : The function to wrap with the save and restore context commands.
val save_to_pngfile : [> `Any ] Cairo.surface -> string -> unit
Saves the given surface to the given file in *.png format. Will overrite a currently existing file.
surface : The Cairo surface to save.
file : The string that represents the file to save to.
val create_surface : int -> int -> Cairo.image_surface
Creates a surface given a width and height.
Returns The Cairo surface.
width : The width of the surface as an int.
height : The height of the surface as an int.
val get_context : [> `Any ] Cairo.surface -> Cairo.t
Gets the Cairo drawing context from a given surface.
surface : The Cairo surface.
val set_color : Cairo.t -> Sig.color -> unit
Sets the current color to paint with.
context : The Cairo context.
val fill_surface : Cairo.t -> Sig.color -> unit
Sets the background color to the given color.
context : The Cairo context.
val polygon : Cairo.t -> Vector.t list -> float -> unit
Draws a polygon by drawing a straight line through each point in the list and then connecting the first and last point.
context : The Cairo context.
pointlist : The list of points
width : The width of the line.
val straight_line : Cairo.t -> Vector.t list -> float -> unit
Draws a straight line through each of the points in the list.
context : The Cairo context.
pointlist : The list of points.
width : The width of the line.
val straight_arrow_line : Cairo.t -> Vector.t list -> float -> unit
Draws a straight line with an arrow on one end.
context : The Cairo context.
pointlist : The list of points.
width : The width of the line.
val point : Cairo.t -> Vector.t -> float -> unit
Draws a point at a given location with a given radius.
context : The Cairo context.
v : The center of the point.
radius : The radius of the point.
val curved_line : Cairo.t -> Vector.t list -> float -> unit
Draws a curved line by taking the point list 4 points at a time. If p,p1,p2,p3 are the points considered, it draws a cubic Bezier spline from p to p3 with p1,p2 as control points. The list of points should look like [p;cp1;cp2;p1;cp3;cp4;p2 ... pn] where cp are control points.
context : The Cairo context.
pointlist : The list of points.
width : The width of the line.
val rectangle : Cairo.t ->
       Vector.t -> Sig.color -> Sig.color -> float -> float -> float -> unit
Draws a rectangle.
context : The Cairo context.
v : The center point of the rectangle.
fillcolor : The color of the inside of the rectangle.
edgecolor : The color of the edges.
width : The width of the rectangle.
height : The height of the rectangle.
linewidth : The width of edge.
val empty_rectangle : Cairo.t -> Vector.t -> float -> float -> float -> unit
Draws a rectangle with a black edge and white interior. Convenience function.

ctrpoint : The center point of the rectangle.
width : The width of the rectangle.
height : The height of the rectangle.
linewidth : The width of the edge.
val fillrectangle : Cairo.t -> Vector.t -> Sig.color -> float -> float -> float -> unit
Draws a rectangle with a black edge and an interior of a given color. Convenience function.
context : The Cairo context.
ctrpoint : The center point of the rectangle.
color : The color of the interior.
width : The width of the rectangle.
height : The height of the rectangle.
linewidth : The width of the edge.
val get_textbox : Cairo.t -> string -> float * float * float * float
Returns a tuple containing x,y,width,height of the given string, were it to be rendered in the given context.
Returns The bounding box of the given string.
context : The Cairo context.
s : The string to get the bounding box of.
val draw_string : Cairo.t -> string -> Vector.t -> float -> unit
Draws a given string at a given centerpoint.
context : The Cairo context.
s : The string to draw.
v : The center of the string.
fontsize : The size of the font of the string.
val rounded_rectangle : Cairo.t -> Vector.t -> float -> float -> float -> float -> unit
Draws a rectangle with rounded edges. Requires a point at the upper-left corner of the rectangle.
context : The Cairo context.
v : The upper-left corner of the rectangle.
w : The width of the rectangle.
h : The height of the rectangle.
r : The radius of the circle sitting at the corners.
width : The width of the line to draw.
