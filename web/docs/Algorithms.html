Module Algorithms


module Algorithms: sig .. end
Contains all of the layout algorithms in functor form. All of the algorithms position vertices and edges with the assumption that the origin is at the top left of the screen with a positive x-direction to the right and a positive y-direction downwards. Some functions are provided to translate from this orientation to the more mathematical orientation of the origin in the bottom left of the screen.
module Graph: Graphtools.VirtualImageGraph
module Digraph: Graphtools.VirtualImageDigraph

module DBounds: Graphtools.FindBounds(Digraph)
val translate_y : float -> int -> float
Translates a y position from an orientation with the origin in the top left corner to a position where the origin is in the bottom left corner.
ypos : The y coordinate to translate.
height : The height of the image.
val translate_x : 'a -> 'a
Translates an x position like translate_y. Does nothing because the orientation switch does not change x coordinates.
val translate_point : int -> Vector.t -> Vector.t
Translates a point from an orientation with the origin in the top left corner to a position where the origin is in the bottom left corner.
height : The height of the image.
