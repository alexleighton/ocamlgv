Module Pack.MakePack.LayeredLayout


module LayeredLayout: Algorithms.Layered(G)(Labeler)
val layout : G.t ->
       int -> float -> float -> float -> float -> int * int * Algorithms.Digraph.t
design g w start_x start_y dx dy Lays out a directed acyclic graph given w number of vertices per layer.
g : The graph to work on.
w : The number of vertices per layer.
start_x : The x location of the first vertex in a layer.
start_y : The y location of the vertices in the top layer.
dx : The change in x between vertices in a layer.
dy : The change in y between layers.
